# 🧠 MindCare

> Plataforma de ligação e gestão entre profissionais de saúde mental e pacientes.

O **MindCare** foi desenvolvido para simplificar o agendamento, acompanhamento e gestão de consultas, oferecendo uma interface intuitiva tanto para médicos como para pacientes. O foco principal é humanizar o atendimento digital e organizar o fluxo de trabalho clínico.

---

## 🚀 Funcionalidades Principais

- **Painel do Paciente:** Visualização de histórico de consultas, agendamento simplificado e área de recursos de bem-estar.
- **Painel do Profissional:** Gestão de agenda em tempo real, prontuário eletrónico e monitorização do progresso dos pacientes.
- **Sistema de Autenticação:** Diferenciação de permissões entre tipos de utilizadores (Segurança de Dados).
- **Interface Responsiva:** Desenvolvida para funcionar perfeitamente em desktop e dispositivos móveis.

---

## 🛠️ Tecnologias Utilizadas

Este projeto utiliza uma stack moderna focada em performance e escalabilidade:

- **Backend:** [Python](https://www.python.org/) com o framework [Flask](https://flask.palletsprojects.com/)
- **Frontend:** HTML5, CSS3 (com foco em acessibilidade) e JavaScript
- **Base de Dados:** SQLite (Desenvolvimento) / PostgreSQL (Produção)
- **Ambiente de Desenvolvimento:** Configurado em **Linux (CachyOS)** com gestão de ambientes virtuais (`venv`).
